# Theory BBQ Lineup

## Purpose

This is the concise index of major physics/foundations targets through the process-first closure-machine lens.

The root doctrine is:

> Information is not enough. Math is not enough. Computation is not enough.
>
> The missing primitive is running closure with admissibility, persistence, receipts, and rendered observer geometry/time.

## Next targets

### Wheeler

Win:

```text
it from bit
information-first physics
black-hole/information instincts
```

Gap:

```text
bit is too static unless embedded in running closure process
information must do work, close, persist, and render
```

Compact line:

> Wheeler found the slogan. We need the machine.

### Tegmark

Win:

```text
reality as mathematical structure
bold refusal to treat human-scale objects as fundamental
```

Gap:

```text
too Platonic
all math exists is not the same as physical process runs
structure is not execution
```

Compact line:

> Tegmark found the library. He did not find the executable.

### Everett / Deutsch

Win:

```text
no collapse magic
relative-state structure
branching as a possible decoherence/closure partition
universal quantum computation pressure
```

Gap:

```text
branching is structure-relative unless an admissible decomposition is selected
Dugic/Francom-style objections press exactly here: mutually irreducible subsystem decompositions can block one clean universal branching story
branching may be a type of closure, but it still needs closure criteria, preferred/admissible structure, persistence, and observer-rendered records
no physical pruning/admissibility rule if everything persists equally
```

Compact line:

> Everett found branching-as-closure, but not the rule that selects which closures count as physical worlds.

Reference hook:

```text
J. Jeknic-Dugic, M. Dugic, A. Francom,
"Quantum Structures of a Model-Universe: An Inconsistency with Everett Interpretation of Quantum Mechanics",
arXiv:1109.6424.
```

### Sorkin / causal sets

Win:

```text
spacetime from discrete causal order
relation/order before continuum geometry
closer to process than most approaches
```

Gap:

```text
causal order still needs closure, persistence, matter, phase, and rendered metric recovery
```

Compact line:

> Causal sets found the skeleton. We need metabolism.

### Jacobson / Verlinde / Padmanabhan

Win:

```text
gravity as thermodynamics/information
not a fundamental force
geometry from entropy/information pressure
```

Gap:

```text
entropy cannot be first if time/process must already be running
heat/exhaust is not the engine
```

Compact line:

> They found heat coming off the machine. We still need the engine.

### Penrose

Win:

```text
spin networks / twistors
geometry and spin structure matter deeply
spacetime geometry alone is not enough
```

Gap:

```text
still too haunted by mathematical objects as primary
needs process, closure, and admissibility beneath the geometry
```

Compact line:

> Penrose saw the shadows were too sharp to be ordinary space. Still too much Plato.

### Smolin

Win:

```text
background independence
time is real
evolving laws / anti-Platonic pressure
strong critique of string overreach
```

Gap:

```text
needs a sharper process/closure machine
needs admissibility and receipt discipline
```

Compact line:

### Rovelli

Win:

```text
LQG
no view from nowhere
relations over objects
```

Gap:

```text
relation still needs running process, closure, persistence, and accounting
```

Compact line:

### Friston

Win:

```text

active inference
persistence through boundary
self-maintaining systems
```

Gap:

```text

```

### Feynman

Win:

```text
path integrals
sum over possible histories
process grammar rather than object-first story
```


Compact line:

> Feynman found the sum-over-possibilities grammar. We need the compiler.
