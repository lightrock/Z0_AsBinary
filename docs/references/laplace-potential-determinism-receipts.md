# Laplace Potential and Determinism Receipts

## Status

This is a sober Pierre-Simon Laplace note for the Maxwell-completion /
closure-machine program.

Laplace did not discover electromagnetism, did not complete Maxwell, and did not
identify the substrate. His useful contribution is different:

```text
Laplace made state, potential, boundary condition, prediction, and deterministic
calculation feel like one scientific posture.
```

That makes him a crucial pre-Maxwell mathematical ancestor.

He belongs near this project because `Z0_AsBinary` keeps asking a Laplacian-style
question in miniature:

```text
given a complete starting object and an explicit rule,
what future orbit, closure, or boundary receipt follows?
```

---

## Historical anchor

Pierre-Simon Laplace worked in celestial mechanics, probability, mathematical
physics, potential theory, and determinism.

The major useful threads here are:

```text
celestial mechanics -> state evolution under law
Laplace equation    -> source-free potential balance
Laplacian operator  -> local curvature / imbalance test
potential theory    -> boundary/source structure encoded in a field-like scalar
probability theory  -> uncertainty as disciplined inference, not mere confusion
Laplace's demon     -> complete state + laws imply complete prediction, in the classical ideal
```

The important point is not that Laplace had Maxwell's field theory. He did not.
The deeper point is that Laplace helped install the idea that physics can be
read as a complete state acted on by lawful update.

## Machine receipt

Through the closure-machine lens, Laplace gives this receipt:

```text
state + law + boundary/source conditions -> determinate evolution or determinate field solution
```

That is a machine-shaped idea.

For a source-free region, the Laplace equation says, schematically:

```text
∇²φ = 0
```

Through this lens, that means:

```text
no hidden local source inside the region;
the interior value is constrained by boundary/source conditions elsewhere
```

That is directly useful for Maxwell completion because field theories often hide
machine questions inside smooth boundary-value language.

The Laplace receipt is:

```text
if the interior is quiet, look to the boundary;
if the boundary is fixed, the interior may be forced;
if the present state is complete, the future may be calculable;
if the result is uncertain, track whether ignorance is epistemic or physical
```

## What Laplace adds

Laplace contributes four pieces of discipline.

### 1. State discipline

Laplace pushes the classical intuition that a complete present state plus the
right laws should determine the next state.

For this repo:

```text
Z0 seed + explicit update rule -> orbit / period / closure receipt
```

This is Laplacian in spirit, even if the rule is discrete rather than celestial.

### 2. Boundary discipline

Potential theory teaches that what happens inside a region can be controlled by
boundary/source conditions.

For Maxwell completion:

```text
boundary conditions are not afterthoughts;
they may be where the machine becomes visible
```

This matters for impedance, detectors, photoelectric transfer, wave reflection,
circuit closure, and `Z0` as an E/H boundary/interface receipt.

### 3. Source discipline

The Laplacian asks where imbalance/source lives.

For Maxwell completion:

```text
is the observed structure generated locally,
forced by boundary conditions,
or inherited from a deeper state grammar?
```

That is a useful control question for both field physics and catalog patterns.

### 4. Probability discipline

Laplace also matters because he did not merely worship determinism. He developed
probability as a disciplined way to reason under incomplete knowledge.

For `Z0_AsBinary`, that becomes:

```text
if we do not know whether a pattern is artifact or receipt,
we need explicit priors, baselines, controls, and likelihood comparisons
```

## Relation to Maxwell

Laplace is upstream of Maxwell in the mathematical atmosphere.

Maxwell did not simply inherit experiments. He inherited a 19th-century style of
mathematical physics where fields, potentials, sources, boundary conditions, and
differential equations had become legitimate ways to talk about nature.

Through this lens:

```text
Coulomb -> static force law
Laplace -> potential / boundary mathematics and deterministic state ideal
Poisson -> sources enter potential equations
Gauss -> flux/source accounting
Faraday -> field lines and relation change
Maxwell -> dynamic field synthesis with displacement current and waves
```

Laplace does not supply the electromagnetic machine. He supplies part of the
mathematical grammar that later lets Maxwell write machine receipts as equations.

## Kitada / Dugić correction: no outside clock for the whole

The companion note [Kitada / Dugić Local-Time Receipts](kitada-dugic-local-time-receipts.md)
adds the missing correction to Laplace's Demon.

Laplace's Demon assumes:

```text
the whole universe can be known at a time
```

But for the universe-as-one, there is no outside clock or outside comparator to
make that sentence operational. Kitada's local-time framework treats time as
local to quantum-mechanical systems, while the total universe has no associated
time. The later Kitada / Jeknić-Dugić / Arsenijević / Dugić minimalist approach
similarly treats time as neither fundamental nor universal, but attributable to
approximately closed systems and subsystems.

Through this repo's lens:

```text
Universe-as-one -> no outside comparator -> no global Laplace clock
subsystem split -> local comparison -> local time / record / change receipt
```

That is the sober physics version of the yin/yang point:

```text
the universe must express at least relational difference for anything like time,
record, or change to be meaningful from the inside
```

Not necessarily two substances. Not metaphysical dualism. But at least two
roles that can be compared:

```text
clock / system
observer / observed
record / recorded
inside / boundary
state / comparator
yin / yang
```

So the correction is:

```text
Laplace gives state + law.
Kitada / Dugić demand local comparator structure.
The Maxwell-completion machine must generate time and records from inside, not borrow them from an impossible outside.
```

## Relation to Wolfram and 't Hooft

Laplace is an older ancestor of both Wolfram and 't Hooft in this project.

```text
Laplace: complete state + law should determine evolution.
Kitada/Dugić: time is local/relational; no global outside clock for the whole.
Wolfram: small explicit rules can generate complex behavior and must be run.
't Hooft: quantum formalism may render a deeper update engine.
Z0_AsBinary: preserve a seed, run a rule, inspect closure, then demand controls.
```

The difference is that Laplace imagines continuous classical determinism, while
this project is open to discrete, informational, boundary, relational-time, or
closure-machine forms.

The useful inheritance is not classical certainty. It is the demand for explicit
state, explicit law, and explicit comparator/boundary structure.

## What Laplace does not finish

Laplace's worldview is powerful, but it can also mislead.

Guardrail:

```text
determinism is not substrate
calculation is not execution
potential is not the machine
boundary solution is not closure proof
complete state knowledge may be impossible or physically ill-posed
whole-universe prediction assumes an unavailable outside clock/comparator
probability may reflect real physical structure, not only ignorance
```

Through the BBQ lens:

```text
Laplace gives the dream of the fully calculable machine.
Kitada/Dugić deny the hidden outside time-comparator for the whole universe.
Smolin reminds us the machine must actually run in time.
Zurek reminds us records and observers are produced, not assumed.
Weyl reminds us description choices can be fake structure.
Shannon reminds us information accounting is not ontology.
Wolfram reminds us some rules must be run.
't Hooft reminds us update engines remain serious candidates.
```

So Laplace is a powerful ancestor, but not the answer.
