# Tesla Apparatus Receipts

## Status

This is a sober Tesla note.

It does not treat Nikola Tesla as a secret-source-of-all-physics figure. It does
not rely on free-energy mythology, wireless-power fantasy, death-ray mythology,
or internet Tesla worship.

Tesla's useful contribution to the Maxwell-completion program is apparatus-first:
he made electromagnetic process visibly run in machines.

Through the process-first / closure-machine lens, Tesla's value is:

```text
rotating magnetic fields
polyphase AC as coordinated phase process
induction motor as field-driven mechanical closure
high-voltage / high-frequency resonance
coupled oscillatory circuits
wireless/tuned-circuit intuition where historically justified
```

That is not fantasy. That is real machine evidence.

## What Tesla did not add

Tesla did not replace Maxwell's equations.

Tesla did not provide a confirmed new fundamental force.

Tesla did not prove limitless free energy.

Tesla did not make experimental controls optional.

The sober statement is:

```text
Tesla did not finish Maxwell at the level of fundamental law.
Tesla built apparatus that made Maxwellian process behavior impossible to ignore.
```

## Rotating magnetic field

Historical contribution:

```text
Tesla's induction-motor work used polyphase alternating currents to create a
rotating magnetic field.
```

Machine receipt:

```text
field relation can be made to rotate as a coordinated phase process without
mechanical commutation at the rotor
```

Maxwell-completion pressure:

```text
phase is not decorative; coordinated phase relations can create real mechanical
work and stable machine behavior
```

Through this lens, the rotating magnetic field is one of Tesla's cleanest
contributions. It is a direct apparatus receipt for phase-coordinated field
closure.

## Polyphase AC system

Historical contribution:

```text
Tesla's polyphase AC patents and Westinghouse deployment helped make generation,
transmission, transformation, and motor use part of one practical AC system.
```

Machine receipt:

```text
electric power is not merely current in a wire;
it is coordinated phase, impedance, transformation, load, and closure across a
network
```

Maxwell-completion pressure:

```text
the electromagnetic machine must explain not only isolated field equations, but
how phase-coordinated networks transmit usable process over distance
```

This is important for `Z0_AsBinary` because impedance is not an incidental unit
word in a power system. It is the name of how a process meets a boundary, a load,
a line, or a medium.

## Induction motor

Historical contribution:

```text
Tesla's AC induction motor used the rotating field to induce currents in the
rotor and produce torque.
```

Machine receipt:

```text
motion can emerge from field relation, induced current, phase lag, and load
closure, without direct electrical contact into the moving part
```

Maxwell-completion pressure:

```text
induction is not merely a classroom law; it is a running closure process where
field change, material response, and mechanical work become one machine
```

This is Tesla at his strongest: not a mystic, but an apparatus builder who made
field dynamics do work.

## Tesla coil / resonant transformer

Historical contribution:

```text
Tesla developed resonant transformer circuits that produced high-voltage,
high-frequency alternating currents.
```

Machine receipt:

```text
coupled circuits can exchange energy through resonance;
frequency, phase, coupling, capacitance, inductance, and boundary breakdown
matter as one process stack
```

Maxwell-completion pressure:

```text
resonance is a closure test: the system accepts some oscillatory relations and
rejects or dissipates others
```

The useful Tesla coil lesson is not spectacle. It is that electromagnetic
systems have admissible modes. They ring, couple, leak, break down, radiate, and
transfer energy according to relation and boundary.

## High-frequency / high-voltage experimental regime

Historical contribution:

```text
Tesla pushed electrical apparatus into high-frequency and high-voltage regimes
where ordinary low-frequency circuit intuition becomes visibly incomplete.
```

Machine receipt:

```text
frequency changes which parts of the electromagnetic machine are visible:
capacitance, inductance, radiation, dielectric behavior, corona, and boundary
```
