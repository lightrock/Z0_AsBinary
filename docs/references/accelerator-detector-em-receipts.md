# Accelerator and Detector EM Receipts

## Status

This is a sober apparatus note for the Maxwell-completion / closure-machine
program.

It does not claim that all fundamental physics is electromagnetism. Particle
accelerators are used to study strong, weak, electromagnetic, and beyond-standard
model processes. The useful point is narrower and stronger:

```text
high-energy particle physics is operationally built on electromagnetic write/read machinery
```

Accelerators use electric fields to add energy to charged particles and magnetic
fields to steer, focus, bend, and store beams. Detectors then reconstruct events
mostly from the electromagnetic consequences of particles passing through matter:
ionization, excitation, scintillation, Cherenkov light, transition radiation,
electron-hole pairs, calorimeter showers, magnetic curvature, and induced
signals.

So even when the target is quarks, gluons, weak bosons, Higgs events, neutrinos,
or dark-sector missing energy, the laboratory interface is still overwhelmingly:

```text
charged matter + electromagnetic fields + material boundary response + recorded signals
```

That makes accelerators and detectors a major machine receipt for this project.

## Machine receipt

Through the closure-machine lens, accelerator experiments say:

```text
prepare charged state
-> accelerate with electric field
-> steer/focus with magnetic field
-> collide or scatter
-> let final-state particles cross material boundaries
-> convert passage/deposition into EM/material response
-> amplify and digitize detector signals
-> reconstruct event from tracks, showers, timing, missing energy, and conservation laws
```

The important receipt is not simply that accelerators use EM hardware. It is that
modern particle ontology is experimentally accessed through a chain of
admissible electromagnetic interactions and records.

```text
field configuration -> beam state
beam state -> collision boundary
collision boundary -> final-state traces
material response -> detector signal
signal custody -> reconstructed event
reconstructed event -> particle claim
```

In this sense, the detector is not passive. It is the apparatus layer where a
hidden event becomes a durable receipt.

## What accelerators add

Accelerators contribute preparation discipline:

```text
electric field -> controlled energy injection
magnetic field -> controlled trajectory and momentum bookkeeping
RF cavity -> phase-synchronized acceleration
beam pipe/vacuum -> boundary conditions for preserving beam state
collider ring / linac -> repeated or directed update path
luminosity -> event-rate bookkeeping
```

That matters because it makes the experiment a running process rather than a
static measurement.

The beam is not merely observed. It is prepared, phase-synchronized, steered,
focused, collided, and counted.

## What detectors add

Detectors contribute receipt discipline:

```text
tracking detector -> charged-particle path and momentum from curvature
calorimeter -> energy deposition through electromagnetic / hadronic showers
scintillator -> excitation converted to light
silicon detector -> electron-hole pairs converted to charge signal
Cherenkov detector -> thresholded radiation receipt from charged particle speed
muon system -> penetrating charged-particle continuation
missing energy -> inferred invisible state from visible conservation bookkeeping
```

The deep lesson is:

```text
particle claims are reconstructed from records, not seen directly
```

That plugs directly into the Zurek side of the BBQ lens: objectivity requires
stable copied receipts. In the accelerator world, those receipts are detector
signals with calibration, redundancy, geometry, timing, and conservation-law
custody.
