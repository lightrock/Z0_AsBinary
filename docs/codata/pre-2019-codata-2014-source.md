# Pre-2019 CODATA 2014 Source Copy

This is the versioned local source table used by the repository pipeline.

Source: NIST CODATA 2014 Fundamental Physical Constants, Complete Listing, archived as `allascii_2014.txt`.
Canonical URL: https://physics.nist.gov/cuu/Constants/ArchiveASCII/allascii_2014.txt

Evidence-chain rule: this file is the named/value/unit source. The generated binary files must be rebuilt from this table by `tools/codata/build_codata_docs.py`, not hand-edited.

This first committed source is the Z0-focused anchor set needed for the pre-2019 impedance experiment. The fetch/parse tools are intentionally separate so the complete NIST all-values table can be pulled into the same pipeline without changing the binary conversion rules.
